To run this network on web ui -
1. Open terminal in the following directory "Charity-Chain-Network"
2. Make sure that docker and docker-compose is installed on your system.
3. Type "sudo docker-compose up".
4. After successful execution of the command open your browser and type "http://10.50.0.2:22004/"
5. Quorum maker with 5 nodes will be opened with the transaction ledger
